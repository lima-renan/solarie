## Getting Started

To run the Eyesome ecommerce web application locally, please follow these steps:

1. git clone https://github.com/SandhyaR1007/eyesome-react.git
2. cd eyesome-react
3. npm install
4. npm start

## Technologies Used

- ReactJS
- React Context API + useReducer
- Tailwind CSS
- Razorpay payment gateway

## Features

- Home Page

- Product Listing Page

- Filters
- Search

- Cart Management

- Wishlist Management

- User Profile Page
- Address Management

- Checkout Page
- Payment gateway integration

- Authentication

- Responsive

- Loaders and Toasts
